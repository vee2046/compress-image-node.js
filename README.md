# compress-image-node.js
node.js compress image 

## 安装
1. 安装`graphicsmagick`，[官网](http://www.graphicsmagick.org)
**注意系统的差异，本文的测试系统是windows 10**

**如果安装前打开了命令行容器，建议安装完后重新打开命令行窗口**
2. 运行`npm i`

## 运行测试
`node index.js`或'npm start'

得到的结果是![image](./result.png)

## 文件结构
| index.js      起动文件
| image.js      功能类
